# Besondere Weihnachten – statisches Dashboard

Dieses Paket ist **sofort bereit** für GitHub Pages.

## Schnellstart (ohne Git, nur Web-Oberfläche)
1. Gehe zu https://github.com/new und lege ein neues Repository an, z.B. `besondere-weihnachten` (Public).
2. Klicke auf **Add file → Upload files** und lade **index.html** hoch.
3. Gehe zu **Settings → Pages**.
   - **Source:** `Deploy from a branch`
   - **Branch:** `main` / `/ (root)` → **Save**
4. Nach 1–2 Minuten erscheint die URL oben, z.B. `https://USERNAME.github.io/besondere-weihnachten/`.

## Öffnen auf dem Handy
- Öffne die GitHub-Pages-URL in **Safari/Chrome**. Die Zahlen und Diagramme erscheinen sofort.
- Tipp: Du kannst die Seite als App auf dem Homescreen speichern.

## Alternative (Repository `USERNAME.github.io`)
Wenn Du das Repo **`USERNAME.github.io`** nennst, ist die Seite direkt unter `https://USERNAME.github.io/` erreichbar.
Lade dazu **index.html** ins Root des Repositories.

## Bearbeiten der Daten
Bearbeite die Daten im `<script>`-Block in `index.html` (Array `rows = [...]`). Committen → Seite aktualisieren.
